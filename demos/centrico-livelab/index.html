Architecture overview

The system demonstrates end-to-end thinking:

  • Data Ingestion: Fetches real-time data from CityBikes and Open-Meteo APIs into PostgreSQL (raw JSONB storage)
  • Feature Engineering: Pandas-based transformation pipeline for structured feature extraction
  • Training: Docker-based trainer that reads from DB, trains scikit-learn pipeline, outputs versioned artifacts
  • Inference API: FastAPI service loading model.joblib + metadata.json, serving predictions via /predict
  • Monitoring: Prometheus metrics exposure (/metrics) with Grafana dashboards
  • CI Pipeline: GitHub Actions with unit/contract tests + synthetic training + model quality gate (F1 >= 0.60)

Inference API

Production-ready FastAPI service:

  • GET /model: Version + feature schema
