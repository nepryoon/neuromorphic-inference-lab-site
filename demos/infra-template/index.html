ML Infra Template

A reusable template for production-minded ML repos: quality gates, Docker, CI/CD, and release/provenance. This is the “portfolio backbone” that makes the rest of the demos credible and scalable.

View Source (GitHub) Evidence: CI/CD Evidence: Docker Back to Demos

What it includes (planned baseline)

  • Standard repo layout (api / ui / data / docs).
  • Quality gates: formatting, linting, unit tests.
  • Docker-first execution (local parity).
  • CI pipeline: build + test + artifact conventions.
  • Release tags + provenance pattern (what code produced what output).
This template will be reused by RAG Copilot / Tabular Risk / Forecast Studio to keep them consistent and “operable”.

How to use (workflow)

  1. Start a new demo repo from this template.
  2. Implement the minimal feature + tests.
  3. Ship via CI and tag releases to lock provenance.
