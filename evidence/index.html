RAG Copilot

Question answering over a curated knowledge base with citations. Includes a live API and a web client.

  • Demo page: /demos/rag-copilot/
  • Repository (app): nil-rag-copilot
  • Repository (infra): nil-infra-template
  • API base: api.neuromorphicinference.com
curl -s https://api.neuromorphicinference.com/health
curl -s https://api.neuromorphicinference.com/version
curl -s -X POST https://api.neuromorphicinference.com/rag/ask \
  -H "Content-Type: application/json" \
  -d '{"question":"How is this deployed?","top_k":4}'

Tabular Risk

A compact tabular scoring demo with a transparent local mode and an optional API mode.

  • Demo page: /demos/tabular-risk/
  • Repository: nil-tabular-risk

Optional endpoint (if deployed): POST /tabular/score on api.neuromorphicinference.com.

Forecast Studio

A forecasting workflow over a numeric series. Local mode runs in the browser; API mode is optional.

  • Demo page: /demos/forecast-studio/
  • Repository: nil-forecast-studio

Optional endpoint (if deployed): POST /forecast/predict on api.neuromorphicinference.com.

Visual Inspector

Image upload with lightweight local inspection. An optional API mode can be used for deployed inference.

  • Demo page: /demos/visual-inspector/
  • Repository: nil-visual-inspector

Optional endpoint (if deployed): POST /vision/predict on api.neuromorphicinference.com.
