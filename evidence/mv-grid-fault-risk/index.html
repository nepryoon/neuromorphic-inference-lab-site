CI training workflow (MLOps)

Artefact versioning + deployment readiness

Model artefact published as a GitHub Release asset. The API fetches it on boot via MODEL_URL, enabling clean promotion of model versions.

Model serving (FastAPI) + live endpoint

Dockerised FastAPI service deployed on Render. Provides `/predict` for online inference and `/docs` for interactive testing. The website demo calls the live endpoint to score risk in real time.
